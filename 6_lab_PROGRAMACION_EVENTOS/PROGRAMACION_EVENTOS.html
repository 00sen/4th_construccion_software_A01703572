Que es material design?


Material design es un lenguaje diseniado por Google.
Este se enfoca en los aspectos visuales para el sistema
operativo de Andriod, aunque tambien es una normativa que
se aplica a distintas paginas web y plataformas.
Podria decirse que se toma como una guia para le disenio de
plataformas web y aplicaciones facilitando la interaccion
de una persona y un dispositivo.
