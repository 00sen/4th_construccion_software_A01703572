Preguntas lab 4


Que diferencias y semejanzas hay entre Java y JavaScript?
- La principal diferencia siendo la conexion entre
HTML y javascript, este realmente no forma parte de
la plataforma de JAVA. Javascript no crea aplicaciones que
se ejecutan en una maquina, y javascript se ejecuta solo
en un navegador web. El codigo JAVA necesita compilacion
y el de Javascript esta todo en texto.
Creo que el principal parecido son algunas cosas
del codigo, la sintaxis.

Que metodos tiene el objeto Date? (Menciona al menos 5)
- getDate: Devuelve el dia del mes segun la hora local.
- getDay: Devuelve el dia de la semana segun la hora local.
- getFullYear: Devuelve el anio en formato de cuatro digitos
segun la hora local.
- getHours: Devuelve la hora segun la hora local.
- getMilliseconds: Devuelve los milisegundos segun la hora local.
- getUTCDay: Devuelve el dia de la semana segun la hora universal.

Que metodos tienen los arreglos? (Menciona al menos 5)
- map: Permite recorrer el arreglo y modificar los elementos
retornando un nuevo arreglo.
- filter: Recorre el arreglo y retorna un nuevo arreglo
con los elementos que pasen cierto filtro.
- find: Recorre el arreglo y regresa la primer coincidencia
del elemento que se busca
- sort: Ordena los elementos en el arreglo de forma ascendente.
- every: Itera el arreglo y retorna true si todos los elementos
pasan cierta condicion, de lo contrario regresa false.

Como se declara una variable con alcance local dentro de una funcion?
- Con let.

Que implicaciones tiene utilizar variables globales dentro de funciones?
- Se podra acceder a ellas desde fuera de la funcion, podran ser
alteradas incluso desde el html que se despliega en el navegador
causando fallas de seguridad y posibles errores.
