Preguntas CSS lab 3


Como ingeniero de software, cual es tu recomendacion
sobre el uso de !important en un CSS?
- Evita que si en algun momento se sobreescribe un valor,
que si se intenta modificar algo o que por algun error algo
cambia, se mantenga lo que se desea.

Si se pone una imagen de fondo en una pagina HTML, por
que debe escogerse con cuidado?
- Puede causar problemas para visualizar informacion,
dificulta la presentacion de toda la pagina.

Como ingeniero de software, cual es tu recomendacion al
elegir las unidades de una propiedad de estilo entre
%, px y pt?
- Para una pantalla es mejor usar px y % para utilizar
pedazos de la pantalla o pixeles. Para imprimir puedes usar pt.

Por que el uso de una version minimizada del CSS
mejora el rendimiento del sitio?
- Optimiza considerablemente los tiempos de carga
de los archivos permitiendo un mejor rendimiento de la pagina.

Preguntas


Cual es la diferencia entre internet y la world wide web?
- El internet es la conexion entre las computadoras
alrededor del mundo y la WEB son las paginas que funcionan
arriba de dicha red.

Cuales son las partes de un URL?
- Protocolo: Como se transfiere la informacion.
- Subdominio: lo que aparece antes del primer punto como www.
- Dominio: nombre que identifica la pagina.
- Ruta: subpaginas y paginas dentro de la pagina.
- Parametro: despues del ?, indicadores dentro de la pagina.
- Etiqueta: Permite hacer scroll hasta un elemento especifico.

Cual es el proposito de los metodos HTTP: GET, HEAD,
POST, PUT, PATCH y DELETE?
- GET: solicita la representacion de un recurso en especifico.
- HEAD: es la primera de las dos partes de un documento HTML.
- POST: inserta o actualiza un recurso oya desplegado.
- PUT: sustituye un recurso por completo.
- PATCH: aplica modificaciones parciales a un recurso.
- DELETE: quita un recurso del servidor original.

Que metodo HTTP se debe utilizar al enviar un formulario HTML,
por ejemplo cuando ingresas tu usuario y contrasenia en algun sitio?
Por que?
- Se utiliza el metodo INPUT, el cual puede ser usado como tipo
contrasenia para que el texto se vea reemplazado con puntos.

Que metodo HTTP se utiliza cuando a traves de un navegador web
se accede a una pagina a traves de un URL?
- Se utiliza el metodo GET, el cual le pide al servidor
los recursos que debe desplegar en la pantalla.

Un servidor web devuelve una respuesta HTTP con codigo 200.
Que significa esto? Ocurrio algun error?
- No ocurrio ningun error, es una respuesta del servidor
dando a entender que puede responder a la peticion del usuario
sin problema.

Es responsabilidad del desarrollador corregir un sitio web
si un usuario reporta que intento acceder al sitio y
se encontro con un error 404? Por que?
-No, ya que el error indica que la pagina que esta solicitando
el cliente no se encuentra en el servidor. A menos de que la haya
solicitado mediante un boton en la pagina, en dado caso si se debe
eliminar ese boton.

Es responsabilidad del desarrollador corregir un sitio web
si un usuario reporta que intento acceder al sitio y
se encontro con un error 500? Por que?
-Si, ya que el erorr indiga que el servidor encontro un problema
interno que le impide completar la peticion.

o desaprobado (deprecated)? Menciona algunos elementos
de HTML4 que en HTML5 esten desaprobados.
-Significa que la etiqueta ya no se usa mas, hay reemplazos

-CENTER
-FONT
-S

Cuales son las diferencias principales entre HTML4 y HTML5?
-HTML5 ofrece nuevas etiquetas para estructura reemplazando
etiquetas como DIV. Nuevos elementos como AUDIO y VIDEO,
integra graficos vectoriales, almacenamiento local en el lado
del cliente, etc.

Que componentes de estructura y estilo tiene una tabla?
- Primero TR y TD para las filas y las celdas. Otras etiquetas
serian TH para encabezados, CAPTION para subtitulos, TBODY
para el cuerpo inicial y TFOOT para el pie de pagina de panera
separada.
- En estilos puedes mover anchura, alineacion, bordes, relleno,
fondo, disenios, lineas, etc.

Cuales son los principales controles de una forma HTML5?
- BUTTON, CHECKBOX, FILE, HIDDEN, IMAGE,
PASSWORD, RADIO, RESET, SUBMIT y TEXT.

Sobre el ciclo de vida y desarrollo de los sistemas de informacion:
Cual es el ciclo de vida de los sistemas de informacion?
Cual es el ciclo de desarrollo de sistemas de informacion?

(voy a responderlas juntas)

- Planificacion: Se define el alcance, se justifica y se elije una
metodologia. - Analisis: Se definen los requisitos que regiran el sistema.
- Fases de disenio: Se define como el sistema
cumplira con los requisitos.
- Fase de desarrollo: El inicio de la produccion.
- Fase de integracion y pruebas: Se comprueba que el sistema
cumple con los requisitos necesarios.
- Fase de implementacion:Se instala el software y hardware, se
capacita a los usuarios.
- Fase de mantenimiento: Se atiende el sistema conforme surgen
necesidades.
